## Project Scope

A freemium SaaS learning platform combining spaced repetition with AI-powered personalization. Users create and study flashcard decks across any subject. The app targets individual learners (language learning, test prep, professional development) as the primary audience, with secondary B2B potential (team onboarding, teacher mode).

**Platform:** Web-first (desktop + mobile responsive), potential native mobile later.

**Core functionality:**
- Flashcard deck creation (manual + AI-generated)
- Spaced repetition scheduling (FSRS-6 algorithm)
- AI-driven adaptive learning (Learning Fingerprint)
- Multi-modal card formats (text, image, audio)
- Gamification without streaks (Depth Score, Learning Wrapped)

---

## Challenge / Opportunity

Existing flashcard apps (Anki, Quizlet) are fundamentally *testing* tools with a learning narrative painted on — they rely almost exclusively on spaced repetition while ignoring four other proven memory consolidation mechanisms: emotion, novelty, storytelling, and sensory context.

The opportunity: build an app where AI silently models how each user uniquely learns and delivers knowledge through narrative and emotion rather than rote repetition — creating a product that improves the longer it's used and is difficult for competitors to replicate.

---

## Design Goals

**Functional:**
- Cold start deck — users experience the product before signing up
- Reverse paywall — users experience AI features before hitting the paywall
- Learning Fingerprint — silent AI profile that adapts the entire experience per user

**Experience:**
- Zero shame / no performance anxiety (eliminate visible scores)
- Feels effortless — "UX so good it disappears"
- Rewards meaningful learning over daily logging (Depth Score, not streaks)
- Respects real user time constraints

**Business:**
- Drive freemium-to-paid conversion through demonstrable AI value
- Enable viral sharing via Learning Wrapped and team duels
- Sustainable solo-dev SaaS with zero infrastructure cost at launch

## Constraints

- **Solo developer** — architecture must be maintainable by one person
- **Tech stack**: Next.js, Supabase (free tier), Vercel (free tier), Stripe, Azure OpenAI (secrets provided)
- **Infrastructure cost**: $0/month at launch — free tiers + Azure LLM
- **MVP scope**: Focus on core differentiators first (Reverse Paywall, Anti-Streak, Cold Start, Shapeshifter Cards, Learning Fingerprint data model)
- **Scale trigger**: Upgrade to Supabase Pro (~$25/mo) and Vercel Pro (~$20/mo) only when traction justifies it
